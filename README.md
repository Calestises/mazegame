# 迷宫游戏项目说明

这是一个使用 C 语言编写的终端迷宫游戏，当前主程序文件为 `mazegame.c`，可执行文件为 `mazegame.exe`。

项目特点：
- 使用随机 DFS 生成迷宫
- 共 3 个关卡，后续关卡地图更大
- 玩家需要先收集钥匙，再到达门的位置通关
- 地图中包含金币和传送门
- 支持保存和读取存档
- 支持 Windows 终端中的 UTF-8 中文和简单颜色显示

## 文件说明

- `mazegame.c`：当前最终版源代码
- `mazegame.exe`：当前主程序
- `mazegame_test.exe`：测试编译版本
- `mazegame one.c`：第一天练习版本
- `mazegame two.c`：第二天练习版本
- `save.txt`：游戏存档文件

## 运行方法

直接运行 `mazegame.exe` 即可。

如果需要自己编译，可以使用 GCC：

```bash
gcc -Wall -Wextra -std=c11 -finput-charset=UTF-8 -o mazegame.exe mazegame.c
```

## 基本操作

- `w`：向上移动
- `a`：向左移动
- `s`：向下移动
- `d`：向右移动
- `p`：保存游戏
- `o`：读取存档
- `q`：退出游戏

## 通关规则

- 每一关都需要先拿齐该关的钥匙
- 拿齐钥匙后走到门的位置即可进入下一关
- 第 3 关通过后显示 `You Win!`

## 地图符号

- `█`：墙
- `@`：玩家
- `D`：门
- `K`：钥匙
- `$`：金币
- `O`：传送门
