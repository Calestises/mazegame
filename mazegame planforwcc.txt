功能：
1.移动方式：wasd移动
2.多关卡设计：预计设计3个关卡
3.道具与机关：钥匙，陷阱夹子，金币，传送点
4.通关判断：获得道具钥匙，达到终点开门即为通关
5.失败判断：拥有生命值，被敌人击中，或者被陷阱命中扣除生命值，生命值归0视为失败
6.优势：游戏进度可保存

目前初步来讲打算先制作一个字符版本的，道具与机关均用字母代替，后续会优化为图片版本

实现方法
1. wasd移动
   读取字符
   根据输入计算新坐标
   如果目标位置不是墙，就更新位置

2. 多关卡设计
   准备 3 张地图
   每张地图对应不同敌人类型
   通关后 level++，加载下一关

 3. 道具与机关
   玩家移动到某格时判断内容
   遇到钥匙：hasKey = 1
   遇到金币：gold++
   遇到陷阱：hp--
   遇到传送点：改玩家坐标

 4. 通关判断
   玩家到达门的位置时
   如果 hasKey == 1，则通关
   否则提示“没有钥匙，无法开门”

 5. 失败判断
   玩家初始有生命值，比如 3
   被敌人碰到或踩陷阱时 hp--
   当 hp <= 0 时游戏失败

  6. 保存进度
  至少保存这些：
   当前关卡
   玩家位置
   生命值
   金币数
   是否持有钥匙
   步数
   当前地图状态
  如果地图会变化，比如金币被捡走、钥匙被拿走，就最好把整张地图一起保存到文件。
  
 保存数据时，保存到新文件里 
  主要用以下两个函数来实现	
  int save_game();
  int load_game();
  保存时：
   fopen("save.txt", "w")
   用 fprintf() 写数据
   fclose()
  读取时：
   fopen("save.txt", "r")
   用 fscanf() 读数据
   fclose()
